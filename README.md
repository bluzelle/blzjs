**blzjs** is a Typescript/JavaScript library that can be used to access the Bluzelle database service.

# Installation



```
yarn add bluzelle
or
npm install bluzelle
```
```
import { bluzelle } from 'bluzelle';
or
const { bluzelle } = require('bluzelle');
```
